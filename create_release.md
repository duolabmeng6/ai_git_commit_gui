### 🚀 升级方法

#### 使用 uv tool (推荐)
```bash
uv tool upgrade git-ai-commit-gui
```

#### 使用 pip
```bash
pip install --upgrade git-ai-commit-gui
```

#### 从源码升级
```bash
git pull origin main
uv sync
```
